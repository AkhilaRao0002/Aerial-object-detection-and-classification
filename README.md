This project demonstrates a deep learning-based approach for classification of aerial objects into two categories: birds and drones, in real-world scenarios. The primary objective is to develop an accurate and efficient system that can distinguish between birds and drones, which is crucial in applications such as security surveillance, wildlife monitoring, and airspace safety. The methodology in this project involves preprocessing image data, including loading, resizing, normalization, and augmentation, followed by the development of a Convolutional Neural Network (CNN) model for classification. In addition, transfer learning layers of different models such as mobilenetV2, resnet50 and efficientnetB0 are utilized to enhance model performance by leveraging pre-trained architectures. For additional functionality, YOLOv8 is incorporated to perform real-time object detection, enabling the system to not only classify but also locate objects within images. The trained models are evaluated using performance metrics such as accuracy graphs, loss graphs, precision, recall, and F1-score to ensure reliability and generalization. Finally, the solution is deployed using a Streamlit-based web application, providing an interactive interface for users to upload images and obtain predictions along with confidence scores. Overall, this project demonstrates the effectiveness of deep learning in solving real-world aerial object classification problems such as Aerial Surveillance, Wildlife Monitoring, Security & Defense Applications and highlights its potential in improving automated monitoring and decision-making systems.
